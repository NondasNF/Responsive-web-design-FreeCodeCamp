Web Development Introduction

The path to becoming a junior in Web Development can sometimes be difficult and the feeling of being lost is a constant, at first the more you learn the more it seems that there is much more to learn and that it is too long a way to go

In this article is separated in topics the initial steps to become a good junior

The first step is to learn how to code, this is done by learning how to write code in a language that is familiar to you, in this case HTML, CSS and JavaScript

HTML + CSS + JavaScript = Web Development
